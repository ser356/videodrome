🎬 letterboxd-cli

Recomendaciones de películas basadas en tu Letterboxd, búsqueda de torrents y streaming BitTorrent integrado (TUI en Rust).

Ver en GitHub →

Instalación

macOS · Homebrew

brew tap ser356/tap
brew trust ser356/tap
brew install letterboxd-cli
brew install --cask vlc

Windows · one-liner (PowerShell)

irm https://ser356.github.io/letterboxd-cli/install.ps1 | iex
Instala Scoop si falta, añade los buckets necesarios, compila el CLI desde código fuente y trae VLC como dependencia. Requiere PowerShell normal (no Administrador).

Linux · cargo

cargo install --git https://github.com/ser356/letterboxd-cli
sudo apt install vlc

Nix / NixOS

nix profile install github:ser356/letterboxd-cli

Uso

letterboxd-cli

La primera vez se te pedirá login de Letterboxd (usuario + contraseña). Todo lo demás va bakeado.
