🎬 videodromev1.3.0

Recomendaciones basadas en tu Letterboxd + búsqueda de torrents en múltiples providers (YTS, Apibay, Knaben, Torznab) + streaming BitTorrent embebido en VLC. GUI (Tauri + React) y CLI/TUI en el mismo binario.

Instalación

Un solo comando · macOS y Linux

curl -fsSL https://ser356.github.io/videodrome/setup.sh | bash
Detecta el sistema, instala Homebrew si falta (macOS) o descarga el tarball en ~/.local/bin (Linux). En macOS resuelve VLC automáticamente como dependencia del cask; en Linux te recuerda el comando para tu distro. Es un bash normal — descárgalo primero si prefieres inspeccionarlo.

Windows · one-liner (PowerShell)

irm https://ser356.github.io/videodrome/install.ps1 | iex
Instala Scoop si falta, añade los buckets necesarios, descarga el binario prebuilt (~30 s). Trae VLC como dependencia. Requiere PowerShell normal (no Administrador).

Instalación manual

macOS · Homebrew Cask

brew tap ser356/cask https://github.com/ser356/homebrew-cask
brew install --cask videodrome
Copia Videodrome.app a /Applications (aparece en Launchpad) y crea un symlink videodrome en /opt/homebrew/bin. La app no está firmada por Apple — la primera vez ejecuta xattr -cr /Applications/Videodrome.app.

Linux · tarball

curl -sL https://github.com/ser356/videodrome/releases/latest/download/videodrome-v1.3.0-linux-x86_64.tar.gz | tar -xz
sudo mv videodrome /usr/local/bin/
sudo apt install vlc
Linux solo tiene CLI/TUI por ahora (sin bundle GUI).

Uso

# GUI: doble click en Launchpad / Start Menu

# CLI (mismo binario):
videodrome recommend --count 30 --min-rating 4.0
videodrome torrents "Alien" --year 1979
videodrome tui                        # TUI en terminal

La primera vez la GUI te pide login de Letterboxd. Todo lo demás (TMDB, OpenSubtitles) va bakeado en el binario.
